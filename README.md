# TPs Machine Learning — Exercises

Hands-on machine learning exercises implementing common algorithms and workflows.

Contents
- Data preprocessing and feature engineering
- Regression and classification notebooks
- Model evaluation and comparison

Installation
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Run
```bash
python TD1.py
# or
jupyter notebook
```
